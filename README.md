# Portfolio personnel

## Description
Site web statique servant de portfolio.  
Il centralise les projets, compétences et informations utiles dans un format clair et accessible.

## Objectif
- Présenter des projets informatiques de manière structurée
- Fournir une vitrine professionnelle simple
- Servir de support pour candidatures et évaluations académiques

## Contenu
- Page de présentation
- Section projets
- Section compétences
- CV
- Contact

## Technologies
- HTML
- CSS
- JavaScript  
(évolutif)
